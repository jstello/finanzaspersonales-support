Privacy Policy

This app processes SMS messages on-device to identify and organize personal financial transactions.

  • Data collected: SMS content required for transaction parsing, transaction metadata (amount, date, provider), and user categories.
  • How data is used: To display transaction history, analytics, and categorization features within the app.
  • Storage: Data is stored locally on your device in the app database.
  • Sharing: Data is not sold. Data is not shared with third parties except where required by law.
  • Permissions: SMS/notifications/contacts permissions are used only to provide app features.

If you have privacy questions, contact REPLACE_WITH_SUPPORT_EMAIL@example.com.

Data Deletion

You can delete app data at any time by:

  1. Using in-app data reset/clear options (if enabled), or
  2. Uninstalling the app, or
  3. Going to Android Settings -> Apps -> FinanzasPersonales -> Storage -> Clear Data.
